Sales Data Analysis Project using Excel and Python  
مشروع تدريبي لتحليل بيانات المبيعات باستخدام Excel وPython

 Tools Used
 Excel: Pivot Tables, Charts, Conditional Formatting
 Python: Pandas, Matplotlib, NumPy
 Objectives 
 Identify top-selling products and regions 
 Visualize monthly sales trends 
 Build a simple dashboard to present insights

Files Included
- `sales_data.xlsx`: Raw sales data
- `analysis_script.py`: Python code for data analysis
- `dashboard.png`: Sample visualization output

This project is part of my data analysis learning journey as a Business MIS student preparing for local and international opportunities.
